¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías

La etiqueta "ul" en HTML5 es una herramienta esencial para crear listas desordenadas en un documento web. Se utiliza para presentar una serie de elementos relacionados que no tienen un orden específico.
Ej: Listado de cantantes del pop

  • Katy Perry
  • Harry Styles
  • Rihanna

¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

La etiqueta "ol" funciona para enumerar la lista o viñetas y si se utiliza el atributo "style" las viñetas son enumeradas alfabeticamente
Ej: Los mejores equipos de futbol

  1. River
  2. Chacarita
  3. Independiente

¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

El elemento HTML representa una lista descriptiva. Con el uso del elemento "dt" se encierra el titulo del elemento y con el uso del elemento "dd" se encierra una descripcion de este mismo.

¿Como te llamas?
Me llamo Dulce Ana Vigna Coronel

¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

El elemento de li declara cada uno de los elementos de una lista. Siempre va acompañado del elemento "ul" o "ol"
Ej: utilizando con el elemento "ul"

  • Mouse
  • Pantalla
  • Teclado
  • Torre

Ej: Utilizando con el elemento "ol"

  1. A
  2. B
  3. C

¿Qué es una url absoluta?

La URL absoluta incluye todas las partes de la URL( protocolo, servidor y ruta) por lo que no se necesita mas informacion para obtener el recurso enlazado

¿Qué es una url relativa?

Las URL relativas describen la ruta a un archivo relativo al actual documento
